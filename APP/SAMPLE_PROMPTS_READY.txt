================================================================================
HOW TO USE THESE PROMPTS IN THE DEMO
================================================================================

1. Open TalentPilot chat at: http://localhost:3000/chat

2. You'll see sample prompts displayed on the chat start screen

3. Either:
   a) Click on any pre-displayed sample prompt, OR
   b) Copy-paste any prompt from this file into the chat input box

4. Press Send (or click the send button)

5. TalentPilot will analyze your organizational data and provide insights

6. Ask follow-up questions naturally - the conversation will continue

TECHNICAL NOTES
================================================================================

- These prompts work with the knowledge base CSV files already integrated
- Responses are powered by IBM WatsonX (kwaipilot/kat-coder-pro:free)

- All responses are non-sensitive demo data from CSV files
